■ NPC별 인형뽑기 아이콘 (투명 PNG)

경로: lorebox-worker\public\icons\npc\
파일명: {iconId}.png  (영문·숫자·_ 만, 예: hanso.png)

권장: 96~128px, 배경 투명

■ URL (S급 결과 예)
?theme=arcade&grade=S&item=한소 인형&iconId=hanso

■ 파라미터 우선순위
1) iconId / npc / icon_id  →  /icons/npc/{id}.png
2) icon=🧸                 →  이모지 1글자
3) png=1                   →  /icons/s-grade.png (공용 S급)
4) 없음                    →  등급 기본 이모지

■ 라벨 목록: labels.json 참고 (로어북·AI용 id 표)
